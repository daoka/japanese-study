A Guide to Japanese Counters

はじめに / Introduction

日本語では、ものを数える時に数字の後ろに「助数詞（じょすうし）」という言葉をつけます。この助数詞は、数えるものの形や種類によって変わるのが大きな特徴です。例えば、英語では "two apples", "two birds" のように数字は同じですが、日本語では「りんご二つ」「鳥二羽」のように数え方が変わります。このガイドで、代表的な数え方を学びましょう。

In Japanese, when you count things, you add a special word called a "counter" (助数詞, josuushi) after the number. A key feature is that this counter changes depending on the shape and type of the item. For example, in English you say "two apples" and "two birds" with the same number, but in Japanese, the way you count changes: "ringo futatsu" (2 apples), "tori niwa" (2 birds). Let's learn some typical counters with this guide.
